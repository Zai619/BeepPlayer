### 使用方式
你可以透過滑鼠點擊不同的琴鍵來產生對應的音，也可以透過鍵盤操作，鍵盤上的ASDFGHJK分別對應到琴鍵上的八個音
透過鍵盤能讓你使用起來更順手

### 特殊功能
按下琴鍵右邊的播放扭，就會播放小蜜蜂的前奏，這時琴鍵還是可以使用喔~
可以讓你練習和其他音樂合奏
